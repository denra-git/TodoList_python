# TodoList Python

A simple and efficient Python-based todo list application to help you manage your daily tasks.

## Description

TodoList Python is a command-line application that allows you to create, view, and manage your tasks. It provides a straightforward way to keep track of what needs to be done.

## Features

- ✅ Create new tasks
- ✅ View all tasks
- ✅ Mark tasks as complete
- ✅ Delete tasks
- ✅ Simple and user-friendly interface

## Installation

1. Clone the repository:
```bash
git clone https://github.com/denra-git/TodoList_python.git
cd TodoList_python
```

2. Make sure you have Python installed:
```bash
python --version
```

## Usage

Run the application:
```bash
python main.py
```

Follow the on-screen menu to:
- Add a new task
- View all tasks
- Mark a task as done
- Delete a task
- Exit the application

## Requirements

- Python 3.6 or higher
